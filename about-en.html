About PerfX

Our mission: professional-grade performance testing, in the browser

PerfX focuses on browser-based CPU, GPU, RAM throughput, Browser I/O, and Network stress tests, with contextual tiers so scores are meaningful.

Start testing View ranking

Methodology

  • CPU multi-core: Web Worker threads + floating-point/matrix-like math; multi-pass max; calibrated toward Cinebench R23 scale.
  • CPU single-core: main-thread latency-sensitive mix (branch prediction + array access); independent calibration for single-core band.
  • GPU: WebGL heavy fragment shader + multiple draw calls; uses `EXT_disjoint_timer_query` if available to bypass V-Sync limits.
  • RAM Throughput: large `Uint8Array` copies with longer duration to reduce cache bias.
  • Browser I/O: IndexedDB large-block write/read; warns if read speed looks cache-inflated.
  • Network: Cloudflare endpoints for latency/jitter + 5MB download; shows “Server: Cloudflare CDN”.

Calibration & Baseline

  • CPU 100%: Apple M3 / Core i5-13600K ≈ 23,000 (Cinebench R23 equivalent).
  • GPU 100%: RTX 4060 / RTX 3060 Ti ≈ 10,600 (3DMark Time Spy equivalent).
  • `HARDWARE_TIERS`: 2025 dataset; nearest-score match to reduce tier jumping.
  • Recent-score logic picks closest tier for stability.

Data & Privacy

History and local leaderboard are stored in your browser’s localStorage only; no server upload. Exports (JSON/PNG) are generated locally.

Tech Stack

Frontend: HTML5 + Tailwind CSS. GPU: WebGL (+ `EXT_disjoint_timer_query`). CPU: Web Worker. Charts: Chart.js. Export: html2canvas.

Best Practices

  • Plug in power / performance mode; avoid battery saver.
  • Keep the tab focused during GPU tests.
  • Browser I/O read spikes may be cache; UI will flag if suspicious.

FAQ

  • Why do GPU scores vary by browser/driver?
    WebGL optimizations differ; `EXT_disjoint_timer_query` availability affects FPS measurement.
  • Why separate single-core and multi-core?
    Single-core stresses latency/branch prediction; multi-core stresses throughput via threads.
  • Are scores identical to Cinebench/Time Spy?
    We calibrate to those baselines, but sandbox, thermals, and power policies introduce variation. Treat as an approximate reference.
